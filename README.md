# Housing Prices Analysis

This repository is dedicated to the coursework and project development for **LIS705: Introductory Analytics for Decision Making**. The project focuses on exploring and analyzing a housing prices dataset to understand market trends and key value drivers.

---

## 📂 Project Structure

*   `data_exploration.ipynb`: The primary Jupyter Notebook for initial data analysis (Assignment 1).
*   `housing.csv`: The core dataset containing house characteristics and prices.
*   `reports`: Directory for project documentation and final reports.

---

## 🚀 Assignment 1: Data Exploration (Current Status)

The initial phase of the project involves getting acquainted with the dataset, checking data quality, and performing basic statistical summaries.

### 📊 Dataset Highlights
*   **Total Records:** 545 entries.
*   **Key Variables:** Price, Area, Bedrooms, Bathrooms, Stories, and Parking.
*   **Data Quality:** 0 missing values detected; all key quantitative variables are of `integer` type.

### 🔍 Research Questions
1. How does house area influence housing prices?
2. Do houses with more bedrooms tend to have higher prices?
3. Does the presence of air conditioning affect house prices?
4. Does the number of parking spaces increase the value of a house?
